# Warm-Up Exercise

> Print a Tree's values in "reverse" breadth-first-search order.

## Feature Tasks

Given this tree:

```text
        {a}
    {b}     {c}
  {d}     {e}  {f}
```

Your function should print:

```text
f
e
d
c
b
a
```

## Requirements

Do not use any built in reversal methods or notation.
